Para ejecutar el codigo: python3 archivo.py
Para ver mas detalles: python3 archivo.pu -v

Coverage es para saber que lineas de codigo se han ejecutado y cuales no.
Coverage: coverage report archivo.py
Correr Coverage: coverage run archivo.py
Mas detalles: coverage report -m archivo.py
Ver en HTML: coverage html archivo.py
Lanzas un server: python3 -m http.server
Para visualiar: Entrar al puerto y dar click en htmlcov/
